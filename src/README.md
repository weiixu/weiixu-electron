## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
npm run dev --env=test
npm run dev --domain=domain || --domain=domain2

# build for production with minification
npm run build
npm run build:dev || --env=test
npm run build:test
npm run build:live
npm run build:demo
npm run build --env=test --domain=domain

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
